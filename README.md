# 🔍 Log Analyzer

**Инструмент для анализа логов веб-сервера с обнаружением признаков кибератак**

Программа на Java с графическим интерфейсом (JavaFX), которая анализирует файлы логов (например, `access.log`) и находит в них подозрительные строки, характерные для распространённых веб-атак.

---

## ✨ Возможности

- 🔎 **Поиск атак** по списку паттернов (SQL-инъекции, XSS, Path Traversal, Command Injection, сканирование)
- 📂 **Загрузка собственных списков** атак из текстового файла (CSV)
- 🖥️ **Графический интерфейс** на JavaFX
- 📊 **Прогресс-бар** при анализе больших файлов
- 📄 **Подробный отчёт** с указанием:
  - номера строки в логе
  - типа найденной атаки
  - IP-адреса атакующего
  - фрагмента подозрительной строки
- 💾 **Сохранение отчёта** (опционально)

## 🚀 Как запустить

### Требования
- Java 8 или выше
- Git (для клонирования)

### Шаги

1. **Склонируй репозиторий**
   ```bash
   git clone https://github.com/SComik/log-analyzer.git
2. **Открытие jar**
   Открываем программу по пути
  ```bash
    \dist\CheckingLogs.jar
  ````
## 🕹️ Как работать

1. Нажимаем на кнопку "Загрузка файла" выбираем **access.log**

2. Нажимаем на кнопку "Загрузка ворд листа" выбираем **wordList.txt**

3. Нажимаем на кнопку "Проверка"

4. Вывод строк в которых обнаружелась подозрительность
